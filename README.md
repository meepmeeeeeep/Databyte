# Databyte Inventory & Sales Management System

A desktop application for managing inventory, sales, users, and financials. Built with JavaFormDesigner.

## Features

- User authentication and management
- Inventory tracking and item management
- Sales and order processing
- Discount code creation and editing
- Financial overview and analysis
- Resupply and payment management

## Requirements

- Java 17 or higher
- XAMPP
- Maven
- JavaFormDesigner (IntelliJ or NetBeans Extension)

## Setup

1. Clone the repository: git clone https://github.com/yourusername/SCAPE.git
2. Open the project in IntelliJ IDEA or your preferred IDE.
3. Build the project using Maven:
4. Run the application:
   - In your IDE, run the `Main` class.
   - Or use the command line:
     ```
     mvn exec:java -Dexec.mainClass="Main"
     ```

## Usage

- Run XAMPP and start Apache and mySQL.
- On launch, the login form will appear.
- Log in with your credentials.
- Use the dashboard to navigate between inventory, sales, users, and financials.
- Add, edit, or archive items as needed.
- Process orders and manage payments.
- Generate and apply discount codes.

## Notes

- All data is stored locally using XAMPP - Apache + mySQL.
- For any issues, please contact the project maintainer.
